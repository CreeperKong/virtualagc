51

TIME2, TIME1.  Double precision word indicating ground elapsed time.  Used for all timing while the AGC is on.  Zeroed at liftoff (by crew procedure), increment by one bit per centisecond.  Scaled centiseconds/228.

Other

All (or most) other downlink items vary somewhat according to the AGC software version in symbolic name, precision (single or double), scaling, numeric format (decimal, octal, floating point), textual description, and so forth.  Because of the absence of an appropriate Guidance System Operations Plan (GSOP) covering this specific software version, such details cannot be provided here.  Therefore, all downlinked data other than the ones described above are presented in raw (unscaled single-precision octal) form.  Careful examination of the source code for the AGC software version may provide additional detail.  Refer particularly to source-code files with names such as:

  • ERASABLE_ASSIGNMENTS.agc
  • FLAGWORD_ASSIGNMENTS.agc
  • INPUT_OUTPUT_CHANNEL_BIT_DESCRIPTIONS.agc
  • DOWNLINK_LISTS.agc
  • DOWN-TELEMETRY_PROGRAM.agc
