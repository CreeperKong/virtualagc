There is little known surviving Apollo-era documentation of the AURORA program, nor its closest relatives (SUNDIAL and SUNBURST), other than rare reference cards and meager memos.  Consequently, all of the information herein is derived from AURORA source code and analogies to less-closely-related software such as LUMINARY.

1a

I.D. word for this list.  Will contain 003218.

1b

Sync bits.  Will contain 004378.

2-5

AGSBUFF, +1, +1, +2, +2, +3, +3, +4 (note the repetition).  AGSBUFF is a 28-word buffer that hold data used to initialize the AGS.  While only 4 of the words are downlinked here, more are downlinked below at words 7-10, 12-15, and so on. 

6a

AGSWORD. Stores the address of the downlist type that was in effect before emission of the Radar List commenced, and is used to restore that prior downlist type after the Radar List finishes.

6b

RADMODES. The descriptions of the bit positions were adapted from the "AURORA 85 & 86, SUNDIAL Mode Bits" document:

Bit
 Meaning
15
 Set 1 for RR continuous designate.
14
 Set 1 for RR antenna being remoded. I.e., remode requested or in progress. This bit is set whenever RRDESSM or RRDESNB determine that a designate may be performed after a remode has been done. It is cleared at the end of remoding by the REMODE routine.

The presence of this inhibits the GIMBAL Limit monitor. If this bit is set during a monitor reposition (see bit 11), the reposition will be interrupted by RR1AXIS and the remode will begin immediately.
13
 Set 1 for RR CDUs being zeroed.   Set by RR Automode monitor and by RR ZERO. Cleared by RR ZEROSB. Inhibits RR CDU Fail from lighting the tracker fail lamp.
12
 Set 1 for RR antenna in mode 2, set 0 for mode 1.  Set by RR ZEROSB according to trunnion angle: 0 for mode I (|T|<70); 1 for mode II (|T|>110°). Establishes limits for RR LIMCHK. Determines gimbal angles for remode and reposition sequences. Determines MODE A and MODE B setting in RR ANGLES (MODE A are gimbal angles if target sighted in present mode, etc.). May be inverted and reinverted (with rupt inhibited) by RR DESSM and RR DESNB to use RR LIMCHK to see if sighting possible after remode. Permanently inverted by REMODE.
11
 Set 1 for RR antenna out of limit.  Monitor Reposition in progress. This bit is set whenever the gimbal limit monitor finds T, S out of limits established for the present mode. It is set by the MONREPOS routine and cleared on completion of the reposition.

Its presence inhibits further operation of the gimbal limit monitor. If, during the reposition, the remode bit (14) is set (by a routine requesting a remode-designate operation), the reposition is interrupted (REPOSRPT) and the remode is started immediately. If, then, when a designate is feasible and STARTDES wants to initiate it, the presence of this bit says antenna control is in  progress, and the associated tasks will use bits 14, 11, and 10 to begin designation as soon as possible. 

If, during a designate operation this bit appears, the designate is terminated, since motion of the spacecraft may have driven the antenna out of limits or the antenna failed. The designate supervisor is given an error return (ROBADENO).
10
 Set 1 for RR designate or request in progress. This bit is set at STARTDES and cleared at the end of designation, successful or otherwise. If it is present at the end of a reposition, control is given to BEGDES to begin designation, possibly preceded by a remade operation. It is cleared on receipt of data good by the designate job to tell the designate task to go to RGOODEND.
9
 Set 1 for LR range on high scale, set 0 for low scale. Set each time A LT data is read from the LR. Used to detect scale changes so the sample which was attempted during the scale change may be discarded.
8
 Set 0 for LR velocity data failed. N samples could not be obtained from 2N tries. This bit in conjunction with Bit 5 lights tracker fail. Set and reset by data reading routines.
7
 Set 1 for RR CDU data OK, 0 for RR CDU fail — ignore if bit 2 and/or 13 = 1. Holds last sampled RR CDU fail bit. The absence of this and bits 2, 13 (auto & zeroing) lights tracker fail. Set/ reset by RRCDUCHK. 
6
 Set 0 for LR desired in pos 1, set 1 for pos 2. Desired LR Antenna position set by LR POSZ and fresh start. Used to monitor LR for proper position while reading data.
5
 Set 1 for LR alt. data fail. No LR Pas Data (See bit 8).
4
 Set 1 for RR data fail. No RR Data. N samples could not be taken in 2 N tries. Turns on tracker fail (see bits 8, 5).
3
 Set 1 for RR on high scale, 0 for RR range on low scale.  See bit 9.
2
 Set 1 for RR not in AUTO mode, 0 for RR in AUTO mode. Set/reset by RRAUTCHK in T4. If just off initiates RRTURNON (unless someone using RR). Its presence inhibits gimbal limit check. If just on, and some program using RR, (RRUSE) do a prog alarm.
1
 Set 1 for no monitor reposition alarm. Turn-on RRAUTCHK sequence. Set by RRTURNON and reset 1 sec after CDUs zeroed. Inhibits prog alarm in monitor reposition prog. Tested by RRZERO and if set, depends on RRTURNON to go to ENDRADAR after testing RR USE, which shows someone is waiting.

39a

UPLOCK. This is the uplink lock.  Only the least-significant bit is used.  When set to 1, further uplink activity is prevented.

39b

T6LOC. This does not appear to be used in SUNDIAL.  In AGC programs where it is used, it serves to prevent timer 6 interrupts (T6RUPT), which in turn prevents control of the RCS jets.

R3D1 stands for digit one of the third register and VD1 stand for the first digit of the verb display, etc.  For the right character of a pair, bit 5 is the MSB with bit 1 the LSB.  For the left character of a pair, the MSB is bit 10 with bit 6 the LSB.  Bit 11 of some of the DSPTABs contains discrete information, a one indicating that the discrete is on.  For example, a one in bit 11 of DSPTAB+1 indicates that R3 has a plus sign.  If the sign bits associated with a given register are both zeros, then the content of that particular register is octal; if either of the bits is set, the register content is decimal data.

If bits 15 through 12 of DSPTAB+11D are 1000, the last 11 bits indicate the state to which the CMC will command the relays; if bits 15 through 12 are 0000, the last 11 bits indicate the actual state of the relays.  A one indicates that the discrete is on.

46

TIME2, TIME1.  Double precision word indicating ground elapsed time.  Used for all timing while CMC is on.  Zeroed at liftoff and incremented by one bit per centisecond. Scaled centiseconds/228.
